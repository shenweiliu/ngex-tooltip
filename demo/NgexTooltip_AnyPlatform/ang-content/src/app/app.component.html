No Header Offset Add header Offset
Instruction 1

For the page with a sticky header, when there is not enough space between the header bottom edge and host element, the tooltip can be auto switched to the area below the host element if the autoBottomOffset attribute value is set to the height of the header, or overlaid on the header if no autoBottomOffset value is set.

Instruction 2

Vertically scroll the window to have the tooltip host element close or away from screen top. Then hover on the host element to see the tooltip.

Tooltip Host Element
Instruction 3

Drag to reduce the window width to have the host element partially out of the window. Then hover on the host element to see the tooltip.

This can also be conducted both close and away from the screen top.
